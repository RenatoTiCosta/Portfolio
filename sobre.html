Esse site faz parte do meu Portfolio

Objetivo apresentar minhas habilidades e conhecimentos para a empresa Accenture Brasil

O site foi criado utilizando HTML, CSS e JavaScript e contém uma página de apresentação, uma página sobre mim, uma página sobre a minha formação acadêmica, uma página das minhas experiências profissionais, uma página dos projetos que eu participei com estudante, desafios ou projetos individuais, como este aqui. Finalmente uma página de contato.

Sobre o desenvolvimento do site:

Ao abrir o link recebe uma mensagem escrita em javascript, o menu de navegação do site foi criado usando HTML e CSS, com links para cada página do site. Além disso, o site possui um script em JavaScript que pede ao usuário que digite seu nome e exibe uma mensagem de boas-vindas personalizada. Na página de apresentação, eu me apresento brevemente e descrevo minhas habilidades e conhecimentos em tecnologia da informação. Por fim, a página de contato contém informações para que os interessados possam entrar em contato com comigo, como endereço de e-mail e links para o meu perfil no LinkedIn e GitHub. Também inclui uma seção de rodapé com informações de copyright e links para redes sociais. Espero que esta explicação tenha ajudado a esclarecer como eu desenvolvei este site e em dois dias todas as atividades realizadas nele.
